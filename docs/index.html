ABOUT ME

My name is Carlos Megia, a game developer from Barcelona. I love to develop video games and learn new skills. My goal is to join the game industry as a programmer and developer, learn everything I can, and develop the games that we all enjoy and love.

Right now i'm studying a bachelor degree in Video Game Design and Development in CITM (UPC). I mainly learn about programming with C++ but I also learn techniques about game design and art.

The objective of this portfolio is to show all the projects I have worked since I started this journey.
